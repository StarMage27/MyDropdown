My Dropdown

Попытка создать выпадающий лист для Jetpack Compose, который бы выглядел и работал лучше чем простые методы создания такого элемента в Jetpack Compose (С помощью кнопки и DropdownMenu).
Пока-что присутствуют визуальные дефекты, которые ограничивают его использование в реальном приложении.

An attempt at creating a dropdown list for Jetpack Compose, that would look and work better than a simple method of creating such an element in Jetpack Compose (With a button and a DropdownMenu).
For now the visual glitches make it not suitable for use in a real app.
